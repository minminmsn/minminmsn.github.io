# 几个免费大语言模型工具推荐

**一、文心一言**

文心一言是百度出品，只要申请排队通过后就可以使用，对中文比较友好，基本的问题都可以回答，最方便国人使用，不需要特别的设备及方式，申请通过后打开网站https://yiyan.baidu.com就可以使用。

下面是随意体验了一把，很有中国风吧！

![](https://mmbiz.qpic.cn/mmbiz_png/NPliceoGvXeP8mibTuP3OPgrQDgTVk2hNvtEboSL8XKdpiaxicDNPN6UYF5TMlm5C2WtysFjiaVbAnukDAllIgDUsJw/640?wx_fmt=png)

**二、单机版Chat Belle**

ChatBELLE-int4 是一个人工智能聊天机器人，它可以与用户进行自然语言对话。它可以回答一些常见问题，例如天气预报、新闻、股票等等。它还可以进行闲聊，和用户交流。ChatBELLE-int4使用Python编写，使用了自然语言处理技术和机器学习算法。

以下是一些关于ChatBELLE-int4的链接：

- GitHub仓库：https://github.com/LianjiaTech/BELLE

BELLE: Be Everyone's Large Language model Engine（开源中文对话大模型）

### App下载

请见Releases。

各平台对应下载&使用说明请见使用说明。目前仅支持macOS。更多平台即将发布！

模型下载

可以去huggingface下载，ChatBELLE-int4

需要先首先执行ChatBELLE app，会建好一个文件夹~/Library/Containers/com.barius.chatbelle。然后将下载好的模型重命名并移动至app显示的路径。默认为~/Library/Containers/com.barius.chatbelle/Data/belle-model.bin。

模型比较大4.35G需要找个网络比较好的地方下载。

**三、ChatGPT Sidebar**

ChatGPT Sidebar是一个聊天机器人工具，可以嵌入到网站的侧边栏中。它使用OpenAI的GPT-3技术，可以与访问网站的用户进行自然语言交互，回答问题和提供帮助。

谷歌浏览器安装插件即可使用，插件的链接为https://chrome.google.com/webstore/detail/chatgpt-sidebar-with-gpt/difoiogjjojoaoomphldepapgpbgkhkb

![](https://mmbiz.qpic.cn/mmbiz_png/NPliceoGvXeP8mibTuP3OPgrQDgTVk2hNvbAvNXEuVJibODPaVnWMsKGGM5oNKSGPNSrjpiaCBJBOwuhKQvoMEPTeA/640?wx_fmt=png)
