# 戴笠黑暗人生的亮点

这是特务王戴笠系列的第四篇文章，介绍的是戴笠的暗杀组织，以及黑暗人生的亮点，如通过军统暗杀汉奸、打击日本侵略者等。

01

戴笠的暗杀组织

戴笠残忍嗜杀，不仅任意杀害共产党人和爱国民主人士及一切反蒋者，对军统内部成员，只要稍一触犯他和他的帮规，就无生路，其间被秘密处死者，当以千计。

------选自章微寒：《戴笠与“军统局”》，第138页

1933年11月9日，戴笠受命于蒋介石派遣手下去杀害爱国将领吉鸿昌。吉鸿昌最为我们所熟知的就是那句“我是中国人”，没想到没有死于日本人之手，最终却被蒋介石秘密杀害。杀害吉鸿昌被认为是军统早期戴笠在华北地区的重大成功之一。

1933年6月18日，戴笠受命于蒋介石派遣手下去刺杀“中国民权保障同盟”领导人杨杏佛，杨杏佛连中10枪，当场身亡。杨杏佛的被杀后激怒了很多爱国人士。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/NPliceoGvXeNxlwxFRIveibbNoRicibnSB6s6v7gJm9YqqAzNS9DriaynsD7S9JPj622G5BpBB2RHXoVOPHZmqWRYiaw/640?wx_fmt=jpeg&from=appmsg)

宋庆龄女士发表了激动人心演讲：这些人和他们雇来的打手们以为靠武力、绑架、施刑和谋杀，他们可以粉碎争取自由的斗争……但是，斗争不仅远远没有被粉碎，而且我们应当更坚定地斗争，因为杨铨为了自由而失去了他的生命。我们必须加倍努力直至实现我们的目标。

鲁迅先生写了哀悼诗：岂有豪情似旧时，花开花落两由之。何期泪洒江南雨，又为斯民哭健儿。

蒋介石这个时候杀红了眼，只要是反对他的或者不赞同他的人，他都想着秘密除掉。要不是考虑到他的夫人和宋子文会因为他害了他们的姐姐而跟他大吵大闹，宋庆龄差点就被戴笠的得力助手沈醉用汽车撞的伤亡。

宋庆龄毕竟有国母和姨姐的身份，其他人就没这么走运了。据说蒋介石给了戴笠一份暗杀黑名单。 该名单不仅列出了要谋杀的中国共产党领导人，而且还有国民党内威胁最大的竞争者如胡汉民、汪精卫等。

当然暗杀国民党竞争者的任务有些没有成功，蒋介石后不顾舆论的激烈反对又命令戴笠在1934年11月14日暗杀了“中国民权保障同盟”的第二号领导人《申报》的主编史量才。

02

**戴笠与日本间谍“毒蛇”的较量**

1937年淞沪会战，日本妄图三个月灭亡中国，除大规模入侵上海，他们还派出特工对上海进行渗透，试图窃取中国军人作战计划并刺杀主张国际调停的宋子文。

这次较量就在这段时间里发生，军统方面，除了戴笠亲自去上海指挥外，还有一个军统驻上海办事处处长——文强。日本方面是特高课大特务——南本实隆，此人阴险狡诈，心狠手辣，被军统称为“毒蛇”。南本实隆长期在中国进行谍报活动，不仅能说出一口流利的中国话，还对中国社会相当有研究，是一位名副其实的中国通。

戴笠委托文强出面以卖情报的形式和南本实隆进行了七次较量，然后约好第八次会谈。第八次会谈，戴笠已布下天罗地网静候南本，结果文强等人却一直没能等到对方，只好撤退。

如果不是戴笠太贪财，前几次特别是第七次都可以把南本实隆从肉体上消灭掉的。可惜没有可惜，只能说他们都是高水平的特工，都算碰上对手了。

03

**抗日是戴笠黑暗人生的亮点**

戴笠一向很有野心，他不甘心一直隐在幕后。1937年8月13日，淞沪会战爆发，戴笠的机会也来了！

戴笠向蒋介石建议在上海组建游击队配合正规军，该建议得到了蒋介石的首肯。后来戴笠来到了上海找到了杜月笙，成立了苏浙别动队，这支游击队的全名就叫“苏浙行动委员会别动队”。

别动队招募了1万多人的队伍，按照戴笠的要求，他们被编为5个支队和1个特务大队，其中第四支队由戴笠领导下的京沪特工组成，支队长是骨干特工张业。最后别动队分派到沪西、浦东、苏州河一带，配合国民党正规军作战，同时还负责清查抓捕间谍、汉奸等任务。

最终戴笠领导的第四支队任务是渡过苏州河去掩护国民党主力军撤退。这一支别动分队是由军统特工组成的，伤亡最惨重，以致戴笠领导的军统局很长时间都没法恢复元气。

11月13日下午，国民党政府发布所有部队撤离上海的命令。别动队也接到国民革命军事委员会的通知，批准他们撤离。不过别动队的直接上级是戴笠，他没有下命令，队员们就坚持不撤。此时，敌众我寡，日军的火又猛，别动队员又牺牲了不少人。直到11月13日晚上，戴笠发出撤离命令，队员们才全部转移进法租界。

11月14日凌晨，上海南市沦陷。在这场中日大战中，坚守到最后一刻的，不是国民党军队，而是这支不被看好的杂牌军。就是这么一支不起眼的小队伍，连续作战几天几夜，为主力部队顺利撤退做出了重大贡献。

上海沦陷后，戴笠的特工队两千多人几乎全部壮烈牺牲了。偶有幸存者，重新回到军统工作。
