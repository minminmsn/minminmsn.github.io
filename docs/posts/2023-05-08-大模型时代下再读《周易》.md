# 大模型时代下再读《周易》

**引子**

**最近有幸拜读了李硕的神作《翦商：殷周之变与华夏新生》，读后便对商周文化产生了浓厚的兴趣，对商周历史及《周易》有了新的理解。最近AI崛起，****就连陆奇都说他跟不上大模型时代的狂飙速度了，我想有些事情AI还超越不了，比如《周易》。****于是我便买了《周易》和《甲骨文字典》纸质版书籍想借着五一假期好好研读一番，后来一想读万卷书不如行万里路。网上一查原来殷墟和羑里都在安阳，还好安阳不是热门旅游地，说走就走如是驱车五百公里，一路顺畅不在话下，先后参观了中国文字博物馆、河南安阳殷墟、羑里周易博物馆，收获颇丰。至于题目为啥叫万分之一解，因为一万个人有一万个哈姆雷特，同样一万个人有一万个《周易》。**

**周易由来**

《易经》的创作过程大概经历了三个阶段：阴阳概念的产生，八卦创立，重卦并撰成卦爻辞。《易经》夏朝时就有了，它本来是帝王用来占卜算卦的方法，只蕴含数理逻辑的卦画。周朝时叫《周易》，据传卦辞是文王所作，爻辞是周公所作，至于《彖传》、《系辞》、《象传》、《文言》、《说卦》等都是后人所作，这里不表。考古队先后在河南省安阳小屯村殷墟遗址及陕西省岐山县凤雏村周文王故居遗址发现的甲骨文，记录了贞人有关卜问的内容（文王故居遗址地下室的微雕甲骨文可能是文王亲手所刻，毕竟那个时候只有商王才有权力把占卜的内容记录在甲骨上），证明了晚殷初周《易经》只是用来占卜算卦。

**什么是周易**

首先我们来看易字，当然得从甲骨文看起，甲骨文易字是太阳下面三个点儿，什么意思呢，太阳雨，变天了，也就是变化的意思。周易前面多了周字，说明周易商，周要改朝换代的意思。至于后人解释周而复始，周易就是周而复始的改变，虽然更适合今天的语境，可能确实是个巧合。周文王在羑里地牢监狱里面茹毛饮血研究六十四卦的卦意时发现了事物变化的规律，更重要的是发现了事情不是一成不变的，特别是处境不好的时候也有变好的希望，此时他萌生了取代殷商的想法。至于周文王为啥要变更殷商下文会说动机，然后周文王在写卦辞时多少揭露了殷商的残暴以及隐晦的表达了灭商谋略。

**怎样读周易**

《易经》是商朝末年的产物，释读其中的文字，需要参照商人的甲骨文。这里只看卦辞，因为卦辞是周文王所作，他到商朝所见所闻及囚禁羑里在《周易》里面多少有些蛛丝马迹。这里举个例子，比如需卦，全是捕俘技术的。“需”通“须”，字意是等待。这个卦主要是有关设伏和诱敌的技巧的。有孚。光亨，贞吉。利涉大川。初九：需于郊，利用恒。无咎。九二：需于沙，小有言，终吉。九三：需于泥，致寇至。六四：需于血，出自穴。九五：需于酒食，贞吉。上六：入于穴，有不速之客三人来，敬之，终吉。李硕是这样解释的：“有孚。光亨，贞吉。利涉大川。”意为，会有所俘获，很荣耀，举行祭祀，占算的结果吉利；有利于渡过大河。前面三条爻辞有“需于郊”“需于沙”和“需于泥”，分别是讲在郊野、沙地和泥泞中设伏。另外还有剥卦和艮卦描述了怎样处理人人牲及血祭鬼神的全过程，由于文字太过血腥就不在这里引用了，但是更刺激人心的是考古记录的照片，看到那些照片才能真正的感受到什么才是万物为刍狗。

**为啥周要易商**

首先给大家看看几个字：卯、伐、孚，这些字甲骨文的含义分别是把人或物对半儿分开、一个戈砍人的脖子、人手抓住一个儿童当人牲，这些都来自于商的人祭，虽然用的是羌（甲骨文长着角的羊）人，羌人和周人应该是本家，让周人抓自己的本家把他们当成动物一样送给商来人祭，后来更甚至是文王的父亲季历和大儿子伯邑考，这样高级的祭品才能打动天上的诸神，文王一家不知道这样的时代还要持续多久，肯定是不能理解也无法接受。在《诗经·大雅·荡》有所体现，它是一首文王控诉商朝的长诗，在其中，文王讲述了商王朝的强大、跋扈、纵酒、狂暴和喧哗。后来武王继承文王遗志，终于完成了灭商大计，连武王自己都想不到灭商如此顺利和快速，可能其他诸侯国也都受不了商的施暴和凌辱吧！

**吾从周**

吾从周是孔夫子说的，相对于夏和商而言的，至于他老人家为啥会说吾从周，如果你了解了殷那段历史肯定就不难理解了，古人虽然更接近古人但是很多事情被有意无意的掩盖了，但是今天考古事业的发达让我们了解很多被历史掩盖的真相。自周朝建立，人们的记忆里便再也没有了那个血腥、恐怖而漫长的年代，商朝的鬼神血祭文化被周公取消了，周公怕重蹈商的覆辙甚至连酒都禁止了，终日乾乾，如履薄冰，敬鬼神而远之，防止他死灰复燃，至此诸神远去，我们也没了宗教，华夏文明得了新生。

**周易今天的意义**

如果今天把《周易》当成文王的心里历程来读，肯定就浪费了后面一系列古人智慧的心血了，我们还是把他当成事物发展的规律来参考，多读肯定不会犯大错，也不会辜负周公对后人对鼓励。实际上，《周易》的基本原理就是通过不断“变化”的方法，拓展、深化着人们对现实世界和未知世界的认识。
