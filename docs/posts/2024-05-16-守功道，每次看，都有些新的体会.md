# 守功道，每次看，都有些新的体会

## ****三杯水杂谈****——分享文学、历史、财经等方面的所见所闻所思所感。点击标题下蓝字“三杯水杂谈”关注，我将为您有价值、有趣味的分享。****

这是篇老文，现在分享给大家，个中滋味，如人饮水，冷暖自知。现在看来，人闲的时候有闲的好：可以思考、可以把控一切，自己就是自己小宇宙的主宰；人忙的时候有忙的好：非常简单、大脑一片空白，除了休息啥都不想，日子过得还很快，也不会有太多烦恼。

01引子

今天学习小组里面一位大牛发了一份文档《2019家庭红宝书：预防家庭小崩溃》，我觉得这个调查研究挺有意思的。然后转发给了一位书友让他也看看，本意上是想看他对号入座归于哪一种家庭呢，经过一番探讨他觉得他不归于文档中几种类型，最后我说你属于随遇而安型的，他说但愿如此。

回家吃饭的时候我还在想这个问题。我想要是更深入的探讨我可能会追问：能否接受命运的安排，不管遇到什么事情？比如命途多舛；再比如遇到大挫折；又比如上天让你喜欢猪八戒，你会不会喜欢。当然应该还有更极端的例子，虽然那样效果更好，但是也不能过于残忍，毕竟只是为了好奇。

然后思绪万千，饭没有多吃，碗也没有洗，就着一罐啤酒打开了大儿子的小破电脑（他说这是最破的电脑，因为经常在他看动画片关键的时候掉链子自动关机，当然了我用时一切会恢复正正常的），急忙记下脑海中的想法，可是已经忘了不少了。每次都是这样，第一次思考时感觉灵感充沛可以浮想联翩，接着就是再二衰，三而竭了，这次就撮合看吧。

02关于守

从来没有想过与马老师主演的电影名字撞衫，然后今天却遇到了，奈何马老师捷足先登，把好名字占位了，索性调换下顺序，这样反而更贴合我想表达的意思，是不是有点儿妙手偶得之的感觉。

这里八卦下，马老师主演了《攻守道》一人横扫所有门派后，有人就说有钱真的可以为所欲为啊！或许，Maybe，Perhaps真是这样的吧。但是我觉得只看到这一层肯定说太小瞧我们的马老师了，至少我们应该去了解乃至理解这三个字的含义吧。这里确实娱乐了一把，顺便说一句娱乐时又想到了Linux之父Just for Fun年轻时所著的《Just for Fun》，书里表达了他的观点人生的意义是从生存到稳定到娱乐，到了娱乐就算成功了，也算是到头了，需要循环去找下个娱乐点。

写到这里我突然发现这些大佬们确实都有娱乐精神哈！我又想到了另外一个我比较喜欢的青年学者熊逸，他在互联网上隐藏得很深，至今没人知道他的真名，记得他开始写《春秋大义》、《孟子他说》、《周易江湖》时是何其的幽默风趣，后来变得越来越理性思辩了，可是又有什么东西经得起思辩呢，所以后来他写的关于王阳明的《一切心法》就严肃得多了（而且提交了关于阳明先生家庭琐事的资料，一下子让阳明先生有血有肉多了，那些把阳明封神的人肯定接受不了这一版本的王阳明吧），没有把娱乐精神坚持到底，不过他本人已经到了看山还是山的境界了，不需要再好奇了。

能接受任何事情，这一点是通过我小儿子的经历得来的。小孩子难带大家都懂的，在他四五个月的时候我妈抱着时完成了首摔，后来七八个月时我和孩子妈由于各自的疏忽大意都导致过他从儿童椅上摔下来，等于说他几个月内已经摔了三次头，现在大家都不互相埋怨了，因为在每个人手中孩子都受到过小小的伤害。虽然都不严重，但是作为家长还是很担心的，怕孩子哪儿摔坏啊，或者长大不聪明什么的。为此我还伤感了一段时间，后来有一天不知是看了黄勃主演的电影《被光抓走的人》还是什么受到了启发，不管他怎样或者不管发生任何情况，只要我都能接受都能陪着他不就好了，何必担心这些没有发生的事情，不管什么结果我都负责不就好了。想到这里我突然就不惆怅了，我立马就觉得该珍惜时间，该干嘛就干嘛了。

其实我之前经常周期性的或者遇到挫败时会担忧或者焦急，那时候我也会去寻找问题的解答，那时我找的答案是：“子曰，‘君子道者三，我无能焉：仁者不忧，知者不惑，勇者不惧。’”可是这个连孔圣人都做不到，何况我等常人。现在找到这个能接受任何事情的解答真是感觉太爽 了，只是不知道这是和自己和解还是人生之道就是如此？

我觉得应该不是守住初心的守，因为你守住初心是为了方得始终。初是始，是因，是为了终，果才刻意为之的，有功利心在里面，而且不可控。我这里的守，强调的是受（看到没有守和受读音都差不多，不知道是不是巧合），受力的受，接受一切的受：就好像《天龙八部》里面的扫地僧一样，他的守可谓一流，能随便打他都能接受，跟没事一样；或者像《碟中谍6：全面瓦解》中有一段质问假扮拉克的伊森能你的底线是什么，他回答我没有底线（当然这个例子可能不太恰当）；也许像庄子说的：“在蝼蚁，在稊稗，在瓦甓，在屎溺。”。

以上就是我说的守。

03 关于攻

既然能接受一切，为啥还要攻呢，只有守不也能活得坦然自在嘛，简简单单的一生多好。我觉得主要是一种强烈的责任感在驱使着我们，让我们富有同情心来关注广大有情众生的福祉。

用《钢铁是怎样炼成的》中的保尔·柯察金的话说就是：“一个人的一生应该是这样度过的，当他回首往事的时候，不因虚度年华而悔恨，也不因碌碌无为而羞耻。”

用爱因斯坦的话说就是：“我的精神生活和物质生活都依靠别人（包括活着的人和死去的人）的劳动，我必须尽力以同样的分量来报偿我所领受了的和至今还在领受的东西。”

用佛家的思想说就是：“你这一生是经过多少的花花草草、飞禽走兽的供养才能有今天，你难道不应该惜福与回报社会吗？”

用《疯狂的麦克斯：狂暴之路》中战子的话说就是：“见证我！见证我踏入英灵之殿！”（ 当然这个例子可能不太恰当）

顺便说一句，难怪爱因斯坦的世界观这么正，可能是受到质能方程E=mc²的影响，他发现宇宙的天体发光发热的奥秘，质量越大能量越大。相对于我们人来说就是你体重越大那么你的责任也就越大，要不然怎样燃烧你的小宇宙，怎样去发光发热呢？显然这里也揭示了减肥的原理，多付出少获取才是减肥的正道啊。

不多说了扯远了，夜深了，思路已经不清晰了，下面是收尾环节。

所以攻就是选择付出，扛起责任。

以上就是我说的攻。
