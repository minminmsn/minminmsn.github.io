# 胡适，你还不信现在比从前强了吗？

今天大A股继续保持狂飙的状态，这四天的表现实乃进十几年来前所未有的好，就当提前祝大家国庆节快乐了！

如果真要了解胡适，确实得从头到尾看看他的日志！虽然正常人谁写日记啊？但是这里真的是当时他心里的映射，所以想要全面的了解这个人，读他的日记再好不过了。

今天继续读，1952年后三个季度，值得描述的事情分享给大家。

01

**你不信中国现在比从前强了吗**

胡适无疑完全西化了，虽然他是官费留学生，可是他从来没有对生他养他的祖国有过多少的感情和留恋。

1952年4月2日下午胡适在远东学会的波士顿年会上，读了一篇题为《从开放门户到铁幕政策》的短文。

听完胡适的演讲后，立即有两位中国学生（一为赵国钧，一为XXX）站起来质问反驳他，其中一位学生“气”得语无伦次！

其一人质问胡适：“你不信中国现在比从前强了吗？”胡适说：“不。”

那个时候新中国刚成立，百废待兴，没有信心也可以理解，如果是现在呢？现在的中国是五常之一，如果胡适还活着，不知道他还满意否？

之前的中国确实积贫积弱，急需一大批能吃苦真爱国的牛人来带领指导，积极建设，哪些离不开大鱼大肉和美国药的文人怎么会委屈自己留在这里吃苦呢？

只有那些具有赤子之心，不考虑自我得失的牛人大师才能做到吧，出去见过世面的不少，吃苦耐劳的不多，大部分只是功利境界而已。

这些留下了建设祖国的才真正的达到了冯友兰所说的道德境界吧！

02

**被杜威影响了一生的胡适**

1952年6月1日晚上八点半，胡适接到约翰·杜威夫人的电话，说杜威先生今夜七点死了。

胡适说杜威先生的思想影响了他一生。那么杜威先生到底是什么思想呢？这里给大家说道说道。

首先，受杜威思想最深的应该是实用主义哲学，所谓适者生存，不适者淘汰，知其不可为而为之的迂腐思想在胡适这里是没有的。而他改名位适，就是要合时宜，精致的自由的利己主义。作为一个独立的个体，追求这个也无可厚非，所谓人不为己，天诛地灭，只是比那些真正的为国为民的人还是差了不少。

其次，胡适不仅学习了杜威的哲学思想，还学习了他的方法论，将其方法论应用到自己的学术研究中，也就是实验主义。后来胡适的最为知名的名言“大胆的假设，小心的求证”这种注重实证和谨慎求证的精神贯穿了他一生的学术活动。

最后，也有人说胡适并没有读懂杜威哲学，或者说并不全懂，由于本人才疏学浅，也不好妄加评论。

03

**回台过62岁大寿，忆往昔爱国少年**

1952年11月19日，胡适回到台北，机场五六百人欢迎，胡适的日记记录了这一盛况，据说是空前的。

1952年12月17日，胡适62岁大寿，很多人给他写了贺寿的文章。特别是罗家伦先生提到胡适45年前的风采，令人尤其深刻。

“我们的父母，生我育我，我们的兄弟爱我助我，所以我爱他们。难道我们的祖国，保护我，教育我，我们倒可以忘记了他吗？倒可以不爱他吗？所以我第一句话就说国是人人都要爱的，爱国是人人本分的事……”

“男子首宜爱国，方为尽分。”

罗家伦先生认为，让62岁鬓发已斑的胡适先生重温一下少年时代的记忆是对于这位一代学人最好的礼物。

反正胡适把这篇文章全部写进了日记里，肯定是认可的，应该认定胡适先生认为自己是一贯爱国的，但是怎样算爱就不得而知了。
