# 人人都可以拥有的三驾车

01

**三架车是哪三架**

我所了解的三架车最早出现在《法华经》里佛祖释迦摩尼讲的一个著名的比喻，既被称为“三车喻”，又被称为“火宅喻”。

佛祖是这样讲的：从前有一座大宅院一天家里突然着火了，主人和大人们都跑出来了，可是小孩子们不知道火的危险，让他们出来不要玩了，可是他们怎么也不听话。眼看火势越烧越旺，这一家的主人灵机一动，大喊：“孩子们我在院子外面给你们准备了三辆车，一辆牛车，一辆鹿车，一辆羊车，都很好玩的，你们快出去玩吧！”孩子们一听，兴高采烈地跑出了院子，于是得救了。

大家都知道佛祖传法怕世人听不懂，多爱用比喻。这里牛车、鹿车、羊车三辆车，对应的就是大乘、中乘、小乘佛法，具体的就不展开了。

后来去读MBA了，经济学教授经常提三架马车，指经济学上对投资、消费、出口的比喻。不过老师也说了三架车说到底就是一架马车。什么车呢？那就是消费！

没有消费，就没有一切，消费就是满足自己的精神上的或者物质上的欲望，然后才会有目标，才有动力去生产。生产后获得了剩余价值才有能力去消费，所以说经济不好的时候消费肯定不行，与消费相关的生意就特别难做，特别是非必需消费相关的生意。

当然，这些都是大方面的，今天我们不谈个人以外的东西。就个人来讲，要想过得好最好也能找到属于自己的三架马车。

所以，今天这里就为大家介绍下人人都可以拥有的三架车，读完本篇文章，你可能会阔然开朗，原来我本性自足啊，不需要家里有矿也能自己生产啊！

02

**牛车是工作**

对于一般人来讲，工作是必须的。今天北京下暴雨想到一个段子比较应景，说下雨了牛会躲在牛棚，马会躲在马舍，而牛马会想尽办法上班，怕迟到。

对于工作的意义，稻盛和夫在《干法》中有很好的阐释：工作是最好的修行。稻盛和夫认为工作不仅是为了生存、为了温饱，工作能够锻炼人性、磨砺心志，工作是人生最尊贵、最重要、最有价值的行为。努力工作能够造就我们优秀的人格，极度认真的工作能够改变我们的命运，使人生朝好的方向转变。

对于老一辈的人来讲，他们会说你不干你吃什么？干等于吃，不干等于不吃。说来真令人疼心，还真有这样的小视频，一个大叔干活的时候吃馒头，不干活是喝水，他自己说不干活时可以不吃。

在这位大叔这里道理很简单：吃又等于生存，所以没活干就不用吃了。

如今这个环境不知道是人多了还是工作少了，总之好多人35岁后就得被动退休，所以家里没矿的人还是适龄的人还是好好工作好好珍惜工作机会了吧。

03

**鹿车是投资理财**

这里的投资理财，大家别被概念吓住了。所谓大海不辞涓涓细流终成其大，一分钱也算是投资。

这个时候大家首先要区分的就是负债和资产。什么是负债，什么是资产，罗伯特·清崎的《穷爸爸富爸爸》里面说的很清楚，可以作为大众理财的入门书籍。

举个最简单的例子，大部分把房子和车子当成资产，其实他们是负债，除非房子、车子出租了产生现金流了才能算资产。

这里再顺便说下门槛最低的投资理财方法，某宝某通里面，只要上网的朋友应至少有其一吧。里面的存钱就是货币资金，这个比银行的活期存款利率会高不少。

存钱多少不重要，关键是养成存钱的习惯，这个是做好投资理财的第一步。然后随着了解到深入，黄金、基金、股票、期货等慢慢等就可以接触到了，当然这是后话，属于进阶级的产品，这里也不展开说了。

04

**羊车是自媒体创作**

羊车这里是拿自媒体创作来举例，拍照片、制作小视频、写网文哪怕是日记也好。只要你有分享欲，再借助一些好用的工具，对一般人都不太难，而且可以一直做。

只要你的作品足够吸引人，这里强调的是吸引人而非高大上，国内网友有大几亿人，能找到共同的爱好的应该不太难。

只要你能吸引一万人关注你，基本上就可以解决你的一日三餐，足够最低的生活保障了。

这里之所以拿自媒体来举例，一是这个门槛比较低，大多数人只要坚持或者看出其中的长远价值都可以实操；二是，最关键的是这个可以产生睡后收入。

只有睡后收入才能跳出时空的限制，一天二十四小时的时间是有限的，公司坐班方圆一平米的空间也是有限的，做好了睡后收入才能达到梦里啥都有的境界。

05

**把最多的精力投入最好的车**

一般来说收入最高的是牛车，其次是鹿车，最后是羊车。当然这个顺序也不是一成不变的，牛车虽然力气最大也是最幸苦的，如果那天睡后收入占比最大了，基本上就可以财富自由了，就不用为生活而烦恼了。

理论上讲，人人都可以拥有三架车，就看我们是否有双发现他们的眼睛。也希望大家早日开发出工作外的车，这样多辆车并驾齐驱才不会心发慌。

但是，每个人都是独一无二的，特长和优势也不一样，所以我们得学会优势管理。利用好二八原理，把我们最多的精力投入到我们最好的车上，这样才能产生最大的收益。
