《论语》共20篇，总计不到16000字。除了子曰两个字外，“仁”字是出现第二高频的字，共计109次。

仁也是儒家的核心思想，《论语》中多篇都有涉及。这也是孔夫子特别强调的思想。

那么仁都有哪些含义，孔子是怎样对弟子们阐述仁的呢？

且看下节分享。

02 仁的含义

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

首先从正面解释什么是仁。代表的语录有下面这几条：

- 

```
`子曰：“刚、毅、木、讷近仁。”（《子路》）`
```

- 

```
`樊迟问仁，曰：“仁者先难而后获，可谓仁矣。”（《学而》）`
```

其次从侧面介绍仁的表现。代表的语录有下面这几条：

- 

```
`子曰：“唯仁者能好人，能恶人。”（《里仁》）`
```

- 

```
`子曰：“不仁者，不可以久处约，不可以长处乐。仁者安仁，知者利仁。”（《里仁》）`
```

- 

```
`子曰：“知者乐水，仁者乐山。知者动，仁者静。知者乐，仁者寿。”（《雍也》）`
```

- 

```
`夫仁者，己欲立而立人，己欲达而达人。（《雍也》）`
```

最后从反面强调哪些不是仁。代表的语录有下面这几条：

- 

```
`子曰：“巧言令色，鲜矣仁。”（《学而》）`
```

- 

```
`子曰：“不仁者不可以久处约，不可以长处乐。仁者安仁，知者利仁。”（《里仁》）`
```

从正面、侧面、反面几个角度反复解释一个定义的含义，这是后来欧洲哲学家喜欢做的事情。

这样基本上能把语言的功效发挥最大化，也能充分地说清楚。

03 怎样才能成仁

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

历史没有新鲜事。

现在人遇到的问题，古人也大多遇到过。

都说《论语》的主旨是仁，上节也花了大量篇幅从正面反面解释了仁。

那么怎样做，才能成仁呢？

这个问题孔夫子的弟子颜渊早就想到了，孔子是这样回到他的。

**颜渊问仁。子曰：“克己复礼为仁。一日克己复礼，天下归仁焉。为仁由己，而由人乎哉？” 颜渊曰：“请问其目。” 子曰：“非礼勿视，非礼勿听，非礼勿言，非礼勿动。”（《颜渊》）**

颜渊是颜回的父亲，也是孔夫子早期的弟子，但是他的学问比自己儿子可差远了。

孔夫子明确的给他解释“约束自己而合乎礼就是仁”。

然后颜渊继续追问到，有没有具体到条目，也就是方法步骤。

孔子没有告诉他做什么，只是告诉他不做什么。简单讲就是不合礼的不看、不听、不说、不做。

鉴于孔夫子是因材施教，不同的弟子问相同的问题可能得到的答案不一样。我们再来找找还有没有其他的答案。

还真找到了，这里有一句非常著名的名言，基本上人人受用。

仲弓问仁。子曰：“出门如见大宾，使民如承大祭。己所不欲，勿施于人。在邦无怨，在家无怨。”（《颜渊》）

除了这个，还有一个有具体条目的回答。

子张问仁于孔子。孔子曰：“能行五者于天下，为仁矣。”请问之。曰：“恭、宽、信、敏、惠。恭则不侮，宽则得众，信则人任焉，敏则有功，惠则足以使人。”（《阳货》）

最后，再举一个特例，樊迟连续两次向孔夫子请教怎样做才能成仁，孔子两次给的答案竟然还不一样。

樊迟问仁。子曰：“爱人。”

樊迟问仁。子曰：“居处恭，执事敬，与人忠，虽之夷狄，不可弃也。”（《子路》）

可见“日日新，苟日新，又日新”，“士别三日刮目相看”，“人不能两次踏入同一条河流”，世界没有不变的东西，唯一不变的就是变。
