# 我还应该坚持下去吗？因为哲学我的心真的好憔悴了？

```
`我是一个农民，也只有初中文化，但我自小喜欢哲学，辍学之后一边打工养家糊口，一边仍然专研哲学上人生的意义这一重大课题。``如今找到了这一问题的答案：根据现代自然科学和社会科学的发展进行这样的总结——人来之于自然，生之于自然，自然本身就已经定义了人生的意义——人能做什么，应当做什么，不应当做什么，都已经是由自然这些力量的不同存在形式所决定了的。``在人类社会的一切立法指引规范中，都隐藏了这一自然性原理。``做人就要做自然规律允许范围内的事情，不要做自然规律不允许做的事情，天下才得以太平。``价值无处不在，风险也无处不在，人活着的一生，就为这无处不在的价值和风险所左右。``这种价值和风险也是人去实现其人生的价值和意义之所在。``这种价值和风险，也构成自然对人类社会的第一推动力…``但我作为一个社会的底层小人物，我的学思想很难有什么途径得到专家们的认可，并被应用于社会，造福于社会。``我的心好憔悴，老婆也因为哲学上的事情和我吵架，家庭经济也难以为继，大家觉得我还应该坚持下去吗？`
```

由于这个问题比较沉重，所以选个轻快的曲子中和一下。

**我的回答如下**

让我想起了一个人他叫王计兵，外号外卖诗人。他的情形有点儿不一样的地方就是他边干活边偷偷的创作，希望您能了解他的生平，或许对你能有所帮助。

作者小时候写作梦被父亲摧毁，但是他没有埋怨，原来原生家庭在他这里不是事儿！结婚后写作梦被老婆埋汰，但是他没有埋怨，原来共同语言在他这里不是事儿！虽然生活非常艰苦，自己的梦想没人理解，但是他依然坚持，依然感恩，依然善良，原来还能这样生活，一个人活成自己的王，独自耍欢！

王计兵其人

> 本书作者王计兵是一个外卖员，奔跑的行程累计15万公里，相当于沿着万里长城跑15个来回。在城市穿梭的日子里，他看到更多跟他一样为生存奔波的人，外卖员、农民工、保洁员、保姆。他们匆匆忙忙，慌慌张张地奔波，却跑丢了自己的身份：既不能在城市中找到一个落脚之地，故乡和父母又在不断远去。如同一个来自底层中国的行吟诗人，王计兵记录下自己和他们的狼狈，以及作为劳动者的自尊。他写道：我也有自己独立的国度，我沸腾的血，就是我奔流不息的江河，我嶙峋的瘦骨，就是我耸立的山川。这些在赶时间的间隙，写在烟盒上、废报纸上的诗句，是来自劳动现场的民歌，生机勃勃，粗粝尖锐。

最后觉得他的诗句很有味道，分享几首如下：

#### **赶时间的人**

从空气里赶出风
从风里赶出刀子
从骨头里赶出火
从火里赶出水
赶时间的人
没有四季
只有一站和下一站
世界是一个地名
王庄村也是
每天我都能遇到
一个个飞奔的外卖员
用双脚锤击大地
在这个人间不断地淬火

#### **新寺庙**

不能确定，我是不是
第一个跨进寺庙的送餐人
大雄宝殿众神就位
居高临下
只俯视着我一个人
这是一次绝好的机会
如果我许愿
必能额外得到提前兑现
不用夹在长长的队伍里
等待叫号
像某些窗口前众多排队者中的一员
可我并不准备跪拜
时间在催
我还有许多单子需要及时配送
此刻，我才是菩萨
面对众多的许愿人

#### **那个人**

开门的是奶声奶气的孩子
他仰着脸仔细瞅了瞅我
转头喊：妈，那个人回来了
一年未见
儿子还是我的儿子
媳妇还是我的媳妇
只有我
从爸爸变成了农民工
从农民工变成了那个人

**请叫我王计兵**

我不叫兄弟

兄弟在别的城市

我不叫父母或孩子

他们都在乡下

我明明一动未动

名字却跑丢了

你可以叫我：上一个

也可以叫我：下一位

**抽烟的男人**

一个男人点燃了

自己的引信

他皮肤黝黑

太阳晒出了

他身体里的铁

他蹲在一堆乱砖上

让自己变得更圆

他接了一个电话

儿子不愿意读书

要出来和他一起搬砖

这让他的身体一瞬间塞满了炸药

他一根接一根地吸烟

直到露水把他变湿
