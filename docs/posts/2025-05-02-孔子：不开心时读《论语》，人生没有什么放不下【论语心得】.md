# 孔子：不开心时读《论语》，人生没有什么放不下【论语心得】

01 《论语》中的乐

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

《论语》共20篇，总计不到16000字，其中“乐”字共出现过48次，可谓是高频字。

更令人意外的是《论语》全篇没有一个“苦”字，这是和儒释道，乃至基督等里面的独一份。

中国最后一个儒者梁漱溟说儒家的人生观是积极向人的，他始终站在人的立场上看问题，关心的是人本身的问题。

话说回来。那么这么多“乐”字，都代表什么含义呢？为啥会花那么大的篇幅在这上面呢？下面一节我们来详细分析。

02 “乐”字的含义

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

首先是音乐之“乐”（yuè，约 22-23 次），多指音乐艺术或礼乐制度。

代表的语录有下面这几条：

- 

```
`“子语鲁太师乐。”（《学而》）`
```

- 

```
`“子在齐闻韶，三月不知肉味。”（《述而》）`
```

- 

```
`“乐而不淫，哀而不伤。”（《八佾》）`
```

其次是嗜好或动词之“乐”（yào，约 9-10 次） 。表示“以……为乐”的主动选择，或对某种境界的追求。

代表的语录有下面这几条：

- 

```
`“知之者不如好之者，好之者不如乐之者。”（《雍也》）`
```

- 

```
`“知者乐水，仁者乐山。知者动，仁者静。知者乐，仁者寿。（《雍也》）`
```

最后是快乐之“乐”（lè，约 15-24 次）。强调内心的愉悦、道德实践的满足感。

代表的语录有下面这几条：

- 

```
`“有朋自远方来，不亦乐乎？”（《学而》）`
```

- 

```
`“一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐。”（《雍也》）`
```

- 

```
`“发愤忘食，乐以忘忧。”（《述而》）`
```

这也是我想重点为大家分享的，具体在第三节再详细说明。

03 儒者之乐

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

儒者之乐的代表是颜回之乐。

也就是他不管身处何地都能从内心散发出来的平静与安乐。

那么我们可以从正面、反面、侧面几个角度来分析。

正面乐就是孔夫子夸奖弟子颜回。

子曰：“贤哉，回也！一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐。贤哉，回也！”（《雍也》）

孔子首位连续两次夸奖颜回，这真是少见，实际上颜回也是孔子最得意的门生。

贤就是有修养，修养到家了，不是有个贤者模式吗？估计说的就是颜回吧，境界到了就没什么放不下，没什么忧愁。

反面乐就是不忧，什么才能不忧呢？

子曰：“君子道者三，我无能焉。仁者不忧，知者不惑，勇者不惧。”子贡曰：“夫子自道也。”（《宪问篇》）

孔夫子不光解释了不忧，还顺带解释了不惑和不惧。弟子子贡还不忘借机夸奖老师，这正是老师的自我写照啊！

意思是说孔夫子做到了仁德、智慧、勇敢。

最后再从侧面来说，这个举一个北宋五子程颐的例子。

北宋五子中的程颐一生谨慎严厉，有人就问他：“你这样活得累不累啊？”程颐说：“吾日履安地，何劳何苦？”

这种不仅不会苦，甚至还以别人认为的劳苦当着乐的境界，简直有点儿反人类。但是不可否认，人家真的是快乐，真的不会有忧愁。

看了这篇文章后，大家感觉怎么样？会不会开心点？以后忧愁时，不开心时多读读《论语》会快乐起来吧。
