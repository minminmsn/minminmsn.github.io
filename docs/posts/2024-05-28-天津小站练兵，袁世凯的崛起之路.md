# 天津小站练兵，袁世凯的崛起之路

这一期还是延续清末民国的历史名人。现在轮到袁世凯系列了，这是袁世凯系列的第四篇文章，简单介绍下天津小站练兵，袁世凯的崛起之路。

中日甲午战争中清朝与日军作战的几乎都是李鸿章领导下的淮军和北洋水师。战争中淮军溃败，李鸿章也被免去了直隶总督和北洋大臣的职位。当然，袁世凯也受到了影响，他回到了老家赋闲。

由于战争淮军武力受到重创，因此不得不组建新军进行填补。位于天津东南部郊区一个叫作小站的地方，此处原本是淮军的军营所在地，至于选谁负责在此组建和训练新军呢？这时候李鸿章想到了一个人那就是袁世凯，为啥是袁世凯呢？一来他之前在朝鲜有练兵的经验，二来他能很好地执行李鸿章的方针策略。

1895年6月底，刘坤一、李鸿章、王文韶三名封疆大吏都上奏折保荐袁世凯，于是光绪帝下旨命已回籍的袁世凯入京觐见。特别是刘坤一，他对袁世凯赞赏有加，曾在奏折中写道：“胸有大器且见多识广，性情忠顺恳笃，办事合理，乃出色人才。”更赞赏道：“知晓军事之文官无人能及袁世凯。”

这封举荐信到了北京后不久，回乡期间的袁世凯就接到了光绪帝的召见。袁世凯被光绪皇帝召见后于1895年8月底呈送了一封万言条，提出了一个完整的改革纲领，其内容为储才九条、理财九条、练兵十二条、交涉四条，充分体现出袁世凯的改革思想。

袁世凯的主张得到光绪帝及朝廷大臣的支持，1895年12月8日，奕訢、荣禄等王大臣联名奏请派袁世凯督练新建陆军，同日光绪帝予以批准，袁世凯正式入主天津小站，开始用西法编练中国首支新式陆军。

袁世凯来到天津小站后不久在原来5000人的军队基础上又追加招募了步兵和骑兵部队，部队总人数达到了7000人，名字也被改为“新建陆军”，简称新军。

新军在装备和军纪等所有方面都效仿西方，新军不仅招聘标准高而且纪律严明，特别是在硬件装备上，步枪枪、大炮、军服、药品等随身必备的日用品也应有尽有。

1898年，袁世凯上奏北京，报告新军的师团组织已经初期完成，还要扩大规模到步兵8个大队8000人、炮兵2个大队2000人、骑兵2个大队1000人以及工兵1个大队1000人。

当时实际目睹了小站的英国军人也评论道：“在西方人看来也非常完美，是中国唯一的军队。”袁世凯的新军规模虽小，却堪称中国最精锐的武力。
