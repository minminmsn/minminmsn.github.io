《论语》共20篇，总计不到16000字。除了子曰两个字外，“知”字是出现最多的字，共计118次。

如果没有大数据统计，真的不会想到“知”字竟然是《论语》中出现最高频的字。

这里的“知”大部分情况下通“智”，说《论语》是一本智慧之书也不是空穴来风。

那么“知”都有哪些含义，孔子是怎样对弟子们强调“知”的呢？

且看下节分享。

02 “知”的含义

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

《论语》中“知”的含义有几种，其中表示“了解、知道”的意思出现最多，我们先看看这些语句。

- 

```
`“人不知而不愠，不亦君子乎？” （《学而》）`
```

- 

```
`“不患人之不己知，患不知人也。” （《学而》）`
```

- 

```
`“父母之年，不可不知也。一则以喜，一则以惧。” （《里仁》）`
```

- 

```
`“樊迟问仁。子曰：‘爱人。’问知。子曰：‘知人。’” （《颜渊》）`
```

- 

```
`“不患无位，患所以立。不患莫己知，求为可知也。” （《里仁》）`
```

了解特意强调了别人不了解自己，不懂自己没什么大不了的，自己也不要生气。

反而自己要知道别人，特别是父母及身边的人，儒家的核心思想向来是严以律己、宽以待人的，是给人方便，给人希望，给人温暖的。

**其次，“知”表示 “知识”的意思。**

- 

```
`“子曰：吾有知乎哉？无知也。” （《子罕》）`
```

令人意外的是，孔子及弟子们很少直接的讲知识。这也是国人几千年含蓄内敛的基因导致，只说了一次“无知”还是孔子自谦之词。

**最后，“知”通 “智”，表示 “聪明、智慧”的意思。**

代表的语句有下面这些：

- 

```
`“知之为知之，不知为不知，是知也。” （《为政》）`
```

- 

```
`“知者乐水，仁者乐山。知者动，仁者静。知者乐，仁者寿。” （《雍也》）`
```

- 

```
`“樊迟问知，子曰：‘务民之义，敬鬼神而远之，可谓知矣。’” （《雍也》）`
```

- 

```
`“知者不惑，仁者不忧，勇者不惧。” （《子罕》）`
```

其中第一句出现的三个“知”前两个都是“知道”的意思，最后一个通“智”是智慧的意思，不自欺欺人才是智慧的。

另外孔子四十而不惑，由说智者不惑，可见他是妥妥的智者了。

03 怎样增强智慧

![](https://mmbiz.qpic.cn/mmbiz_png/iblb6AQIJ4DIbHS1DJ0S8X4bYj6nafSuPKt4Caxyz7FYZdjOVIo3weKIO5KspF60p2hsdN10zZGuzum5wUop4OA/640?from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_png/FzUFibK07nY7X9rzkLZeDPXMmb39jIHOMLQeSrNKaHWQ3ywPFXcMfU0XKxlKRgb5mMrebWOS4QDSzTKJPOn9xPw/640?from=appmsg)

那么怎样增强智慧呢？

是不是等到四十岁后，人生阅历增加了就自动智慧了呢？

显然不是，不是说男人至死方少年吗？除非这个少年生而知之。

《论语》中提供了多种方法，从学习、深思、实践、反省、参考等步骤全方位的提升个人智慧。

现在我分别总结给大家。

“学而不思则罔，思而不学则殆。”（《为政》）

这是最为人所熟知的吧，学而思缺一不可，只学不思考肯定学不会，只空想不学习也会迷惑不解。

“盖有不知而作之者，我无是也。多闻，择其善者而从之；多见而识之；知之次也。”（《述而》）

孔子说过“生而知之者，上也；学而知之者，次之。”学问需“多见、多闻、多问”，并转化为实际经验，如果不是生而知之者，又不愿意学习，那么要想增长智慧就难于登天。

“三人行，必有我师焉：择其善者而从之，其不善者而改之。”（《述而》）

另外学习还得有选择的学习，近朱者赤近墨者黑，学好的改掉坏的，这样见贤思齐，慢慢的就会变得有智慧。

“吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？”（《学而》）

最后，反省也是不可或缺的，人孰无过，过而能改，善莫大焉。记得之前写流氓大亨杜月笙介绍他的生存法则时，就特意强调了反省的力量。

他每天睡觉前会反思这一天自己说过的话，做过的事情，然后反省，发现其中的疏忽错误，然后加以改变。

这一点做的最好的当然属于孔子最中意的弟子颜回了，说他不贰过。意思是说颜回同样的错误只犯一次，你再也找不到他第二次犯了，这样的人不是圣人谁还是呢！
