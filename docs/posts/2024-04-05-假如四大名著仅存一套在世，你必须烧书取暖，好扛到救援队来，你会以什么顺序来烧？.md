# 假如四大名著仅存一套在世，你必须烧书取暖，好扛到救援队来，你会以什么顺序来烧？

我会以字数多少和页数厚薄来烧啊，肯定先烧厚的后烧薄的，最后竟留下了《三国演义》。

四大名著的字数统计如下：

- 《三国演义》通行版一百二十回，70余万字，含回目字数734，321字。
- 《水浒传》篇幅最长的杨定见序一百二十回本《水浒全传》，共九十六万字。
- 《西游记》通行版共八十二万字。
- 《红楼梦》高鄂、程伟元续写的通行版即人民文学出版社一九八二年八月北京第一版，一九九零年六月北京第八次印刷共一百零七万五千字。

先烧《红楼梦》，红楼梦看着比较伤元气，等待救援的时候肯定得保持体力为主，另外本来就是没有写完的书，既然结尾定不了就不给后人留下悬念了。

然后烧《水浒传》，水浒传里面都是打打杀杀、吃人放火，暴力倾向严重，最后基本上都没啥善终者，留不下来也不可惜。

最后两本看运气了，其实我都想留，如果非要留下一本，那就先烧《西游记》吧，按理说《西游记》师徒四人最终经历九九八十一难成功上岸了，寓意非常好，暗示救援成功。但是我更喜欢《三国演义》，因为里面有曹老板、司马懿、武圣关二爷、卧龙凤雏、孙十万、周瑜、大乔小乔等等数不清的历史名人事迹。

提到关老爷和诸葛丞相，基本上甭说东南亚地区的传播范围了，即使放在全世界范围内也都能挤进东方古代名人榜的前几名。关二爷一人扛两大重担，他代表着中国文化中极其重要的忠与义；诸葛亮则代表着中华两千年儒家文明的人格极致：“鞠躬尽瘁，死而后已。这些都是拜小说《三国演义》所赐，要不然真没有这么高的流传度。

《三国演义》是中国文学史上最伟大的历史演义作品，没有之一。因为《三国演义》不仅主体贴合史料，而且加戏的这个度，拿捏得特别好。三国这帮好汉，当你看完后就会联想，张飞和马超要是再打几百回合谁能赢呢？赵子龙这七出七入太拉风了！但前提是曹操爱才不让放箭。这大白脸够意思！张三爷在当阳桥头那大喝得多大声音呢？貂蝉得有多好看呢？要不吕布顶不住呢！这谁顶得住啊！诸葛亮舌战群儒和骂死王朗时得多过瘾哪！当年的草船借箭得多悬哪……

预知细节如何，可以看各种《三国演义》衍生小说，比如成君忆的《水煮三国》、易中天的《易中天品三国》、渤海小吏的《三国争霸》。
