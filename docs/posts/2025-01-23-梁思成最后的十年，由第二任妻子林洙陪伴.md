# 梁思成最后的十年，由第二任妻子林洙陪伴

*1*

疼失所爱

1955年4月1日，林徽因不幸在北京同仁医院病逝，享年仅仅51岁。

1928年3月21日，梁思成和林徽因在加拿大渥太华结束8年爱情长跑顺利结婚。他们在一起生活了27年，相识了35年。

他们结婚时，梁思成曾问林徽因：“有一句话，我只问一次，以后都不会再问，你选择的人为什么是我？”林徽因的回答是：“答案很长，我得用一生去回答。”

林徽因的生命相对来说虽然很短暂，但是很精彩，以林徽因宁折不屈的性格应该很难熬过后来的日子。

在1944年日军攻占贵州都匀，直逼重庆，重庆城内人心惶惶时，12岁的梁从诫问道：“妈妈，如果当时日本人真的打进了四川，你们打算怎么办？”林徽因若有所思地说：“中国念书人总还有一条后路嘛，我们家门口不就是扬子江吗？”

这里有点儿像存在主义哲学家萨特所说的“人是绝对自由的，哪怕面对死亡”。

*2*

黄昏之恋

今天有一位白发苍苍的爷爷留言说只记得林徽因设计过国旗国徽，不记得梁思成设计过啥建筑物。我回复他说至少有北京香山公园的梁任公墓和八宝山的林徽因墓啊，我还亲自去香山吊唁过呢！

林洙和林徽因同姓而且是老乡，林洙父亲也是搞建筑的，想着子承父业。所以当时看到搞建筑的程应铨就一见如故，并把女儿许配给他，当然这是后话。

当林洙第一次来到清华园，首先见到的是林徽因，等得到林徽因的帮助在清华稳定后才见到了梁思成，他们第一次见面时林洙大概20岁左右。

1955年，林徽因走后，梁思成的日子越发艰难了。不仅靠边站了，还发生了“大屋顶”事件，成为众矢之的，遭遇了惨烈的批判。

痛苦的日子我一般喜欢一笔带过，还好世界给梁思成关了一扇门后又给他开了一扇窗，在梁思成极度压抑和痛苦时林洙走进了他的生活，梁思成就像抓住了救命稻草一样，他和林洙从工作谈到生活，从生活到无所不谈，慢慢的就越界了。

不知道从什么时候开始起，梁思成就习惯了有林洙来家里整理书籍和书信，至少他觉得他和林洙之间产生了感情。总之他们之间的感情，用一个词来形容，那就是日久生情，量变到质变。

梁思成终于忍不住了在一个深夜凌晨2点钟提笔给林洙写了一封深情的告白信并落款“心神不定的成”，说本想把这份超越友谊的感情埋藏在心里，可是还是情不自禁的泄露了，因为已经完全被“俘虏”了。

还好，林洙并没有拒绝这份感情，这时的梁思成谦称自己已经是个“老、丑、怪、残疾”的小老头， 对这份来之不易的感情感到很珍惜。

但是梁思成的子女以及亲朋好友都不这么认为，不是觉得他们之间相差太悬殊林洙另有所图，就是觉得梁思成一把年龄了还娶林洙是多此一举，当然子女也特别反对，毕竟没有那个后妈能取代林徽因的地位。

面对这么多反对声音，梁思成依然固执己见，就像后来钱锺书所说：“老年人恋爱就像老房子着火一样，怎么都扑不灭。”

事后想一想，在那个特殊时期，梁思成可能就是希望有一个人能陪他说说话，照顾他的生活。老年人也需要被爱，被照顾，被关心，被陪伴，所以林洙对梁思成的体贴与温柔，就给了他莫大的慰藉与温暖。

*3*

十年陪伴

在林徽因走后的7年，1962年，61岁的梁思成与34岁的林洙结婚了。

其实林洙此前也有一段婚姻，前夫是程应铨，是吴良镛的同班同学，和梁思成应该是同事加师生关系。

1949年，林洙和程应铨结婚，梁氏夫妇还送过礼，1957年迫于形势林洙主动提出离婚，并带走了两个年幼的子女。

1972年，71岁的梁思成不幸病逝，此时林洙才44岁。她和梁思成在一起生活了十年，但是他们之间已经认识了二十四五年。

林徽因的母亲何雪媛比较长寿，直到90岁才去世，在林徽因与梁思成先后离世后，林洙接替了照顾他的责任，最终她的后世也是林洙料理的。

不管他们两人结合时是多么的不被人看好，有多少人反对，但是从结果上看，林洙并没有对不起梁思成，当然她也没义务对不起其他人。

所以，从这个角度看，晚年的梁思成由林洙来照顾，对梁来说确实是一个不错的选择，要不然估计他在1966年就撑不过去了。

梁思成去世后，林洙并未再嫁，一心扑在收集和整理梁思成遗稿上。在2001年北京人民大会堂举行的“梁思成先生诞辰一百周年纪念会”上，林洙以阅读梁思成先生全集选段作为自己独有的纪念方式。
