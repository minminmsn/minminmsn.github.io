# 唯有好书不可辜负

所谓好书就是能给你解惑、启发、乃至授业传道的书。具体来分又可以分为道和术，简单的讲就是体和用，通俗的讲就是概念和方法，商业的讲就是战略和战术。

我看题主已经列举了不少书籍了（这是回答某个题主的问题后来又转成文章的，因为我好多时候都是即兴发挥的，有时候题主删了问题我连留档的都没有），但是不成体系，都是点连面都不是，所以更难成体系。刚开始的时候我们可以通过题主这样的去收集、去博览，这些都没问题。但是然后得确定自己研究的方向，最后再去扎根深挖。从过来人的角度讲，广撒网虽然可以多敛鱼，但是抓不住大鱼。

题主要的是启发，启发多半儿来自于三言两语，但是这三言两语一定是来自于一流的书籍、高质量的报告、牛人的交谈。对于一般人来讲后两者都很难企及，唯有一流的书籍比较容易得到。

那话说回来，何为一流的书籍呢？首先还是经典，什么是经典呢？肯定是那些牛人一身心血的结晶而且经过多年多少牛人的认可才算。切不可把时间浪费在那些一年出好多本的畅销书作家身上了。

题主已经搜集了不少了，比较符合国人气质的有《易经》、《老子》、《论语》、《庄子》，但是大家会发现这些书读的人很多收获多的人却很少。原因有很多，最主要的是不得其时而读，什么才算得其时而读，这个争议比较大，我的建议就是每隔一段时间读一读。比如一年、三年、五年，甚至十年为间隔，就拿《易经》来说吧，你三十岁读、四十岁读、五十岁读肯定感受是不一样的，而且会发现每次读都能发现新的东西，觉今是而昨非的感觉，这就是经典书的魅力。

如果真的怕曲解了或者入门不了，可以找一些牛人的注释去读，根据古代的牛人更加接近更古代的牛人的思想原则，他们的注释还是很值得参考的，比如郭象的《庄子注》，北宋五子的书，宋代理学集大成者朱熹的《四书章句》等等，他们的注释结合经典原文一起读那效果肯定差不了。

经典是修内功的，内功虽高，人在社会上行走还得学点儿外功。外功就是各种方法论，比较实用的东西。比如你是医生，你不得读《黄帝内经》、《本草纲目》、《伤寒论》嘛；如果你是律师，你不得读《商君书》、《韩非子》、《法经》嘛；如果你是军人，那《孙子兵法》、《吴子兵法》、《六韬》、《三略》不得是你的必修课嘛；最后搞计算机的，那《The C Programming Language）》、《ComputerSystem:A Programer's Perspective》、《Just For Fun》不也得Learning下嘛。

除了经典还有各行各业的专业学术书籍外，还建议多读些历史和文学作品，要不人生多无趣啊！

历史书籍的话最起码的《史记》、《资治通鉴》、《二十四史》这些正史得读吧，其他的野史和再创作就更多了，从上古到先秦再到宋元明清的都有，比如阿菩的《山海经密码》，寒川子的《鬼谷子的局》，孙皓晖的《大秦帝国》，渤海小吏的《两汉风云》、《三国争霸》，成君忆的《水煮三国》、易中天的《品三国》、王觉仁的《大唐兴亡三百年》，阿越的《新宋》，当年明月的《明朝那些事儿》，二月河的《雍正王朝》，最全的当然还是易中天的《易中天中华史全24卷》（上周上课简教授说易中天的父亲是他师爷，那我岂不是可以叫易中天教授为师叔啊）了。有对这方面感兴趣的自己就知道去搜了。

文学这范围就更广了，先说国内的，诗词文就够喝一壶的了。诗有《诗经》、《唐诗三百首》；词有**《宋词三百首》**，又分豪放、婉约、闲情、逸趣；文的话就更广了，为了省事直接上《古文观止》和曾国藩的《经史百家杂钞》，都是名家选的，特别是曾国藩的眼光毒辣，选的都是精品中的精品 。

另外明清小说不得不提，越往后的作品越接近现代人的语言风格和表达方式。四大名著就不说了（如果时间特别有限《三国演义》和《红楼梦》前八十回不可省啊），《金瓶梅》、《三言二拍》、《聊斋志异》切莫错过了。

到了清末民初更是大师云集、那么多大师的文章都不想错过，只需要记住他们的名字然后搜他们的书读就行了，像梁启超、王国维、辜鸿铭、梁漱溟、陈寅恪、鲁迅、林语堂、胡适、冯友兰，后面的还有很多大师就不多说了，从前面的大师文章中也能读到不少，你顺藤摸瓜就能找到更多了。

国外文学我感觉有点儿水土不服，目前最喜欢的还是卡夫卡（打工人的知音，反工作的先驱，有工作的人都应该读读）和加缪，特别是加缪的《局外人》和《西西弗神话》一直激励着我这样特立独行反复折腾的人。

其他的就是哲学方面书籍，从希腊哲学开始，古希腊三哲苏格拉底、柏拉图、亚里士多德肯定不能不读吧，文艺复兴后面的笛卡尔、斯宾诺莎、休谟、康德、黑格尔、维特根斯坦都可以读，虽然后面的哲学家的名著越来越晦涩难懂，感兴趣的了解一下还是可以的。

现在的作家，茅盾文学奖和一些获得国外奖项的作家作品真的很值得我们一读。比如路遥《平凡的世界》、王小波的《黄金时代》、史铁生的《我与地坛》、莫言的《生死疲劳》和《晚熟的人》，刘慈欣的《三体》、刘震云的《一句顶一万句》、毕飞宇的《推拿》。另外好多也拍成了影视作品，更便于我们观看和欣赏了。

以上都是我脑中现在还印象深刻的，其他读了很多记不住不代表他不经典不专著，只是我没有经常读或者还没到触发我记忆的时候，当我们一个人的时候或者困难的时候，这些一流的书会给我们更快的反馈，更多启发，特别是人在困境中读《易经》收效启发最大。

一句话，艰难困苦造就人，读一流的书做一流的人。反之，那些牛人的后代之所以成不了牛人就是缺乏与之匹配的环境，说白了得配得上自己遭受的苦难。

最后的最后我想引用豆豆的小说《遥远的救世主》中女主角芮小丹的一句话作为结尾：只要不是我觉到、悟到的，你给不了我，给了我也拿不住。只有我自己觉到、悟道的，我才有可能做到，我能做到的才是我的。
