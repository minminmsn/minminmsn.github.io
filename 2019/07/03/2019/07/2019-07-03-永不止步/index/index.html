三杯水

永不止步

Created2019-07-03|Updated2023-05-26|wisdom
|Post View:

人到中年容易油腻，只是容易而已，又不是一定会的，是油腻还是清淡？那得看你内心够不够坚定，有没有执念，够不够坚持，我觉得那些顺其自然，全都是放任自流，不足以把你带入宽的境界！世间那一个有所成的人不都是要经过千辛万苦、千锤百炼才得来的，所以凡是容易的都是便宜的，不是真的，最终只会消耗你的时光！

凡世间万法，都由心生，内心光明才无所畏惧！就可以远离失败，并不是说你不会遇到失败，而是说失败已经打不倒你，对失败产生了免疫。

读书、学习、健身都是一辈子的事，日供一卒，一日一进是具体的方法论，坚持实践就行！

大处着眼，小处下手是战略思维与战术方法的结合，缺一不可。大处着眼，眼是眼界，心智；小处下手，手是动手，实践。格局要大，规划要大，突破点小，一步一步去做，日积月累，最终成事。成事后眼界进一步提升，形成良性循环！

什么时候都不能停下来，永不止步！我始终坚信人可以变得越来越好，无论什么都可以学而知之，修炼来提升，如果不信不妨试一试，如果没有效果不妨再坚持坚持。

成人的世界没有喜欢，只有坚持，你想变成怎样就能变成怎样！要是你还没有变成你理想的样子，那就少看手机，多读书，多思考，多锻炼！

人生是一场有去无回的旅行，也是一场不断铸造自己成长的过程，很害怕老之将至回首往事发现都不止一提，实在对不起你吃下去的食物，喝下去的水，呼吸进去的空气，更何况这个世界对你的栽培与扶养呢！
