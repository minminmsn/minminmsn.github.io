Twee Twee Dee

Posted on 2019-06-21 Edited on 2023-05-26 In talk

PS：这是我2015年双12写的，刚看了下感觉以后再也写不出来了，重新发布下！

我是一个武侠迷，从小就看过很多武侠小说，后来也看了很多武侠电影，慢慢的被武侠电影里面的配乐吸引住了。提起香港电影繁盛时期的武侠音乐，我们就不得不提到一个人，他的名字叫黄霑。记得，第一次知道他的名字还是上高中的时候，那时就知道他是音乐鬼才。他一生惊才绝艳、风流不羁，令人又爱又恨。他和写武侠的金庸、写美食的蔡澜、写科幻的倪匡并称为香港四大才子，他曾经用音符和文字为我们创造出整整一代人梦想中的江湖，他是个能把“粗口”变成传说、讲“黄段子”讲到出书的不文文人。

但是造化弄人，2004年11月24日，黄霑一笑西游。真是天妒英才啊，香港的电影与音乐，竟似从此生生被抽走了侠骨柔肠。而且香港的武侠片更是从此没落了，再无创意，后来我也就慢慢调整兴趣到科幻片，当然这是后话。

如今70、80年代的人，大多心底会有种武侠情节，回顾当年港片的武侠歌曲，我们会发现，有气势有意韵有极强江湖味道的歌，已经被前人唱尽了。这是整整一代香港影视和音乐为我们营造的江湖。在这个江湖中，黄霑的地位是不可替代的。

黄霑一生创作颇多，在香港武侠电影的黄金时代，几乎所有的武侠佳作都请黄霑来谱曲，其中最爱的电影笑傲江湖，翻拍我最爱的金庸武侠小说，可见一部佳作都是无数天才的结晶，其结果可谓是经典中的经典，这首插曲就是沧海一声笑。

这首绝唱，说来创作还有点小插曲，黄霑与电影导演徐克是知交好友，徐克90年代初拍的许多古装电影都由他配乐填词。1990年，他受命为徐克的电影《笑傲江湖》谱曲，写了六稿，徐克都不满意，无奈之中，随意翻阅古书《乐志》，看到一句话，“大乐必易”。心想最“易”的莫过于中国五声音阶（宫、商、角、徵、羽），就反用改成“羽、徵、角、商、宫”，到钢琴前一试，婉转动听，声色悠扬，颇具中国古曲风韵，于是就顺着写出了整条旋律，这就是今天你我所听到的《沧海一声笑》了。公认的绝佳作品，和徐克电影的气氛宛如一体，相得益彰。

沧海笑，滔滔两岸潮，浮沉随浪只记今朝。 苍天笑，纷纷世上潮，谁负谁胜出天知晓。 江山笑，烟雨遥，涛浪淘尽红尘俗事知多少。 清风笑，静若寂寥，豪情还剩一襟晚照。 沧海笑，滔滔两岸潮，浮沉随浪只记今朝。 苍天笑，纷纷世上潮，谁负谁胜出天知晓。 江山笑，烟雨遥，涛浪淘尽红尘俗事知多少。 苍生笑，不再寂寥，豪情仍在痴痴笑笑。

随着林青霞的倾城一笑，立马长嘶，宝剑归鞘，绝江湖，入尘世，就这样——梦里逍遥。

整首曲子一泻千里，畅快淋漓，两人儒冠，泛舟泱泱江水之中，黄昏之下，琴声悠悠，于沧海中一声笑，多少凡尘俗世至于胸外，怡然风流，胸襟宽阔，不计得失，相忘于江湖。

一曲笑傲江湖曲，道尽一场江湖事。这种看透世事，与世无争的豪情，甚至让我想起了一首小诗，他就是英国诗人兰德（W.S.Landor）的同名小诗《生与死》：

我和谁都不争， 和谁争我都不屑； 我爱大自然， 其次是艺术； 我双手烤着生命之火取暖； 火萎了， 我也准备走了。 世事无常，无需抑郁，出入江湖，最后，都在沧海一声笑中。
