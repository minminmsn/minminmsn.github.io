人生的意义汇总

Posted on 2019-02-18 Edited on 2023-05-26 In wisdom

最近读了几本书，里面都有意义的观点，所以就汇总了如下，仅供大家参考：

人生有何意义

摘自 胡适《人生有何意义》 生命本没有意义，你要能给他什么意义，他就有什么意义。与其终日冥想人生有何意义，不如试用此生做点有意义的事。

人生的意义在追梦

摘自 殷海光《人生的意义》

胡适说，人生应该有梦，否则人生不是太不丰富吗？现在你们都有理想，但出了社会便可能不同了。那时各奔前程，各种打击，各种现实的考虑，都可能使得你把崇高的理想收敛起来。这就是现实在考验我们的道德力，我们的理想性，我们对真对善对美的追求是否迫切。在世界上每一个角落都是如此的。我们是否能撑得住，就在这个关头。现在是考验我们的时候了。

论生命的意义在奉献

摘自 爱因斯坦《我的世界观》 人类生命的意义是什么？或者总体而言，一切生物的生命意义是什么？要回答这个问题，意味着要有宗教情怀。你可能会问：那么，提出这个问题究竟有什么意义呢？我的回答是：凡将其自身的生命及同类的生命视作无意义的人，不仅是不幸的，而且几乎不适于生存。

生命的意义是爱

摘自 李银河《享受人生：我的人生美学》 生命虽然从宏观看没有意义，但是从微观看却汹涌着各种各样的意义。其中最有意义的不是荣华富贵，不是声明显赫，而是爱。在爱面前，其他的一切不得不黯然失色。

人生的意义是创造

摘自 梁漱溟《人生的意义》 人生虽不好说有目的，但未尝不可说人生有其意义。人生的意义在哪里？人生的意义在创造！

人生本没有意义

摘自 毕淑敏《人生本没有意义》 我说人生是没有意义的，这不错，但我们每个人都要为自己确立一个意义。
