平凡的世界

Posted on 2019-05-12 Edited on 2023-05-26 In note

之前没有读长篇小说的习惯，最近半年才开始花点儿时间读读长篇，到现在为止也就只读了一部半。之前读乔治·R·R·马丁的<<冰与火之歌>>，读了一半儿后发现作者没有计划写完整部小说，所以就终止了，打算靠HBO改编的《权利的游戏》获知结局。最近读了一部就是路遥的《平凡的世界》，总共花了三十个小时左右大概读了一遍。

之所以很少读长篇小说，是因为觉得读小说需要闲情逸致的，否则就是暴殄天物。读长篇小说又是件很费时的事儿，所以选择一本值得读的小说才有可能读下去。所幸《平凡的世界》真是每个中国男人都应该读的小说，在我心中他的地位不亚于马里奥·普佐创作的长篇小说《教父》，而且他更适合国人阅读，更贴合国人的精神面貌及人物性格。国家、集体、责任、亲情、爱情、友情，哪一个都不缺，这些都是作为一个男人都不可缺少的必修课！

绝大部分人的一生都是平凡的，面对平凡且普通的生活是否有底气度过这一生，是否有勇气面对真实的自己，遭遇挫折及打击能否重新找回自信开启新的生活，长达几十年的苦难是否能够抗住，这些疑问我相信读过此书的人都能够找到些答案或安慰。

罗曼罗兰说过：“世界上只有一种真正的英雄主义，那就是在认清生活的真相后依然热爱生活。”我不知道《平凡的世界》中少安、少平兄弟是否认清生活的真相，但是他们无疑是热爱生活的，对生活负责的，最终没有辜负生活，艰苦的劳动能够赶走空虚的大脑，强烈的社会家庭责任感让他们人格得到了升华。

电影《无问西东》中也讲过，如果提前了解了你所要面对的人生，你是否还会有勇气前来？愿你在被打击时，记起你的珍贵，抵抗恶意。愿你在迷茫时，坚信你的珍贵，爱你所爱，行你所行，听从你心，无问西东。这里虽然比较抒情，但我感觉同样适合每一个平凡的我，每一个平凡的我也是一个珍贵的我、独一无二的我。

最后的最后的我想说的是看到田晓霞牺牲了，真的好伤心啊！
