众生皆苦

Posted on 2019-05-09 Edited on 2023-05-26 In talk

这里所说的众生皆苦不是佛经里面的内容，是最近生活的一些体验。 我是搞IT这一行业的，具体是从事运维的工作。原来觉得运维天天加班很苦，最后去了创业公司发现开发也很苦，最近一次紧急项目出差发现销售可能更苦。要是不去发现，不去一线体验，真的很难体会到各个行业的幸苦。
